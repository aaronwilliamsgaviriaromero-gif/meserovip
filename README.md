# meserovip - Premium Nightclub Management

¡Bienvenido a **meserovip**! Esta es la implementación funcional de la visión de lujo para tu aplicación de gestión.

## Características Implementadas:
1.  **Dual Mode (Local vs. Turista)**: Cambia instantáneamente entre precios locales y precios VIP de turistas usando el interruptor dorado en la parte superior. El tema visual cambia de azul neón (Local) a dorado profundo (Turista).
2.  **Gestión de Mesas Visual**: Un mapa interactivo del local. Toca una mesa para empezar un pedido.
3.  **Terminal de Ventas de Alta Gama**: Una cuadrícula fluida con todos tus productos, emojis representativos y precios automáticos.
4.  **Sistema de Carrito Glassmorphic**: Resumen dinámico del pedido con cálculo de impuestos y servicio (10%) en tiempo real.
5.  **Estética Premium**: Efectos de cristal (glassmorphism), fuentes modernas (Outfit) y animaciones suaves.

## Cómo Ejecutar:
Simplemente abre el archivo `index.html` en cualquier navegador moderno (Chrome, Edge, Safari). No requiere instalación de servidores ni bases de datos externas para esta demo funcional.

## Estructura del Código:
-   `index.html`: Estructura principal.
-   `styles/main.css`: El sistema de diseño, temas y animaciones.
-   `js/menuData.js`: Todo tu menú cargado (Whisky, Tequila, Cervezas, etc.).
-   `js/app.js`: La lógica que hace que todo funcione.
